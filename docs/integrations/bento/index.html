BentoML Adapter

Deliver typed PlanLang actions to Bento services with reproducible, vendor-attested builds.

Deployment

praeon adapters enable bento \
  --digest ghcr.io/praeon/adapter-bento@sha256:... \
  --kms-key projects/pilot/keys/bento

Capability

{
  "tool": "bento",
  "name": "invoke",
  "scopes": ["service:inference", "endpoint:claims"],
  "effects": ["service.invocation"]
}

Dry-Run

praeon kernel dry-run \
  --plan ./plan.json \
  --adapter bento \
  --environment sandbox

Audit Trace

Operational traces include adapter digests, request/response hashes, and KMS usage logs. Audit traces redact payloads but maintain semantic proof.
