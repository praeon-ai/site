Ollama Adapter

Vendor-attested adapter for running typed PlanLang actions through Ollama with least-privilege execution.

Enable the Adapter

praeon adapters enable ollama \
  --digest ghcr.io/praeon/adapter-ollama@sha256:... \
  --kms-key projects/pilot/keys/ollama

PlanLang Capability

{
  "tool": "ollama",
  "name": "generate",
  "scopes": ["model:pilot-llm", "output:moderated"],
  "effects": ["content.generated"]
}

Policy Example

allow {
  input.capability.tool == "ollama"
  input.constraints.riskLevel == "low"
  input.steps[0].effects == ["content.generated"]
}

Replay Guidance

Semantic replay runs Ollama in deterministic mode with seeded prompts and adapter logs to validate equivalence.
