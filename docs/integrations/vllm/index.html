vLLM Adapter

Typed PlanLang actions with deterministic replay for vLLM deployments.

Adapter Install

praeon adapters enable vllm \
  --digest ghcr.io/praeon/adapter-vllm@sha256:... \
  --kms-key projects/pilot/keys/vllm

Capability Definition

{
  "tool": "vllm",
  "name": "completion",
  "scopes": ["model:pilot-7b", "output:moderated"],
  "effects": ["content.generated"]
}

Guardrail Policy

allow {
  input.capability.tool == "vllm"
  input.constraints.riskLevel == "medium"
  input.constraints.budget.max <= 100
}

Replay Checklist

  • Record adapter digest and model commit.
  • Store prompt/response pair in operational trace.
  • Replay with sandbox credentials for semantic equivalence.
