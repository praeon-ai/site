Operational & Audit Traces

Praeon emits deterministic evidence for every executed plan. Operational traces remain encrypted; audit traces are redacted and exportable.

Trace schema (preview)

{
  "traceId": "trace_2024-05-12T17:20Z_9f2c",
  "planId": "plan_12345",
  "approvals": [{ "stepId": "s1", "approver": "manager.alice" }],
  "execution": {
    "adapter": "email",
    "effects": [{
      "type": "email.sent",
      "status": "success",
      "metadata": { "to": "user@example.com" }
    }]
  },
  "operational": { "storage": "encrypted", "kmsKey": "projects/pilot/keys/email" },
  "audit": { "redactions": ["body"], "exportedAt": "2024-05-12T17:20:05Z" }
}

Export commands

praeon trace export \
  --plan plan_12345 \
  --format audit \
  --destination s3://praeon-audit/pilot

praeon trace export \
  --plan plan_12345 \
  --format operational \
  --destination kms://praeon-hsm
